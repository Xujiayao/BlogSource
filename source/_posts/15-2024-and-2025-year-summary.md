### 雀魂麻将

2024 年二月份去 [@Pedro](https://pedropo.com/) 家玩的时候，第一次真正学习了麻将的基础知识和香港清章麻雀的玩法。后面就在网上自学，同时了解到雀魂这款游戏（结果就上瘾了）

线下打就是打港麻，线上打就是打日麻 ヾ(≧▽≦\*)o

2024 年 6 月花 600 软妹买了大洋化学 SMART 套装 + 门风盘 + JUNK MAT。线下也能打日麻啦！！！

不过基本还是拿这副麻将来打港麻。日麻还是太偏向专业性和竞技性了，周围没有几个人会打 o(TヘTo)

## 工具

Python 最适合拿来干什么呢？当然是拿来写小工具咯！

不需要编译，实时解释，非常方便拿来写点临时小工具。加上现在有 AI 加成，直接把需求告诉 AI 就好，干活老快了ヾ(•ω•\`)o

记录一下我这次使用的 DeepSeek 帮忙写的 `png_auto_rename.py` 工具，方便我在以后整理年度报告截图时使用：

{% hideToggle 代码较长，为您收藏 %}
```python
import os
import sys

def rename_images_in_folder(folder_path):
    """
    将文件夹中的PNG图片按顺序重命名为01.png, 02.png, ...
    确保至少两位数（如01, 02, ..., 10, 11, ...）
    """
    try:
        # 切换到指定文件夹
        os.chdir(folder_path)
        
        # 获取所有PNG文件并按名称排序
        png_files = [f for f in os.listdir('.') if f.lower().endswith('.png')]
        png_files.sort()  # 按A-Z顺序排序
        
        if not png_files:
            print("文件夹中没有PNG图片文件！")
            return
        
        print(f"找到 {len(png_files)} 个PNG文件，准备重命名...")
        
        # 计算需要的数字位数，确保至少有2位数
        total_files = len(png_files)
        digits = max(2, len(str(total_files)))  # 确保至少有2位数字
        
        # 先记录原始文件名和目标文件名的映射关系
        rename_plan = []
        for i, filename in enumerate(png_files, 1):
            # 生成新的文件名，确保至少两位数，如 01.png, 02.png, ..., 10.png, ...
            new_name = f"{i:0{digits}d}.png"
            rename_plan.append((filename, new_name))
        
        # 显示重命名计划
        print("\n重命名计划：")
        for old_name, new_name in rename_plan:
            print(f"  {old_name} -> {new_name}")
        
        # 询问用户确认
        response = input("\n是否确认执行重命名？(y/n): ").lower()
        if response != 'y':
            print("操作已取消。")
            return
        
        # 执行重命名
        print("\n开始重命名...")
        renamed_count = 0
        
        for old_name, new_name in rename_plan:
            # 检查新旧文件名是否相同
            if old_name == new_name:
                print(f"  - 跳过: {old_name} 已使用目标名称")
                continue
                
            # 如果目标文件名已存在，先添加临时后缀避免冲突
            if os.path.exists(new_name):
                temp_name = f"temp_{new_name}"
                os.rename(old_name, temp_name)
                # 如果原始文件被重命名为临时文件，需要更新old_name
                old_name = temp_name
            
            # 执行重命名
            os.rename(old_name, new_name)
            renamed_count += 1
            print(f"  ✓ {old_name} -> {new_name}")
        
        print(f"\n完成！成功重命名了 {renamed_count} 个文件。")
        
    except FileNotFoundError:
        print(f"错误：找不到文件夹 '{folder_path}'")
    except PermissionError:
        print("错误：没有权限访问此文件夹。")
    except Exception as e:
        print(f"发生错误：{e}")

def wait_for_key():
    """等待用户按任意键继续"""
    if os.name == 'nt':  # Windows系统
        os.system('pause')
    else:  # Linux/Mac系统
        input("请按任意键继续...")

if __name__ == "__main__":
    # 使用方法说明
    print("PNG图片批量重命名工具")
    print("=" * 40)
    
    if len(sys.argv) > 1:
        # 从命令行参数获取文件夹路径
        folder_path = sys.argv[1]
    else:
        # 如果没有提供参数，使用当前文件夹
        folder_path = input("请输入包含PNG图片的文件夹路径（直接回车使用当前文件夹）: ").strip()
        if not folder_path:
            folder_path = "."
    
    # 调用重命名函数
    rename_images_in_folder(folder_path)
    
    # 等待用户按任意键继续
    wait_for_key()
```

## 有关「标点符号」的话题

细心的人一路读下来可能会发现，我的标点符号（如「『引号』」、《书名号》、〈篇名号〉）的使用习惯不符合简体中文的用法。

主要还是因为我从小到大都在香港接受教育。尽管现在写字、阅读时更喜欢较为好写、简洁的简体字，但繁体字才是我更熟悉的「母字」。因此，在书面文章中使用繁体中文地区的标点写法可以说是很自然的。

其实在博客里我一直都是在使用「『』」这种引号形式。今年在写大学文章习作时被中文老师纠正后，才逐渐倾向于使用学术上更通用的写法。在简体中文中，引号多用 “ ”；而在繁体中文书面语中，则多用「」——香港、台湾都是使用这一套的哦~
