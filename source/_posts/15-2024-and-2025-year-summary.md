### 支付宝

支付宝消费确实比较少，基本上一直都在用微信支付。

忘了是去年还是今年，反正内地的银行卡我是这两年刚办。支付宝的服务基本上都是跟银行账户挂钩的，所以对比微信支付，需要用到支付宝的场景还是要少不少的。微信直接好友间转账、收红包，进入「零钱」就好了嘛。不过我这两个月已经在积累转出银行账户里免手续费的「提现免费券」了，毕竟放「零钱」里没有任何利息。

### AlipayHK

AlipayHK 我以前还是经常用的。后来我寻思我都早就有银行账户了，为什么不去办信用卡和借记卡呢？所以后续我都是直接用 Google Pay 支付，就直接绕过 AlipayHK 啦~

此外我有很多张卡嘛，每张卡（当然是免年费的咯）都对应不同品类的消费。每次消费选择对应的卡，可以最大化信用卡返现√

### 地图软件

我是 4 月 1 号愚人节拿到的 C1 驾照。后面就天天开车出去溜达。没开车以前我是只有安装 百度地图 的，后面才新装的 高德地图。所以今年我是两款软件混用的，就放一起说吧ヾ(•ω•\`)o

暑假实习的时候每天上班来回都要开 30 公里的车，算是对沿路了如指掌了。所以我就在每天换着导航软件开，就觉得 百度地图 的「车道级导航」不是太灵敏，而且容易显示到错误的车道（仅管 百度地图 才是首发「车道级导航」的软件）。

有看我上面讲解 2024 年度报告的各位应该都知道了，我在去年就开始用模拟器玩 欧卡2，所以我的车技在没报考之前就是非常棒的（~~自夸~~）。驾校我去也只是随便到处练练和研究，教练都不用管我hhh (。・∀・)ノ

我家里人对我的水平也是非常认可，全部需要开车的场景都是交给我来开。事故集锦看太多了，养成了防御型驾驶，能够对道路上的环境实时分析并提前预判，所以开的车是「稳而不慢」。我爸妈都经常自嘲说他们的车技都快要退化了哈哈哈。目前开了得有接近 10000 公里了，无事故无剐蹭ヾ(≧▽≦\*)o

高德的加油功能还挺不错的，比加油站官方的油价要便宜超级多的！开到外地没油卡的时候，我都会开 高德地图 来搜索附近支持 高德加油 的加油站，再去。接近用油卡的价格了，大约 9 折到 95 折的样子！

## 工具

Python 最适合拿来干什么呢？当然是拿来写小工具咯！

不需要编译，实时解释，非常方便拿来写点临时小工具。加上现在有 AI 加成，直接把需求告诉 AI 就好，干活老快了ヾ(•ω•\`)o

记录一下我这次使用的 DeepSeek-V3.2 Thinking 帮忙写的 `png_auto_rename.py` 工具，方便我在以后整理年度报告截图时使用：

{% hideToggle 代码较长，为您收藏 %}
```python
import os
import sys

def rename_images_in_folder(folder_path):
    """
    将文件夹中的PNG图片按顺序重命名为01.png, 02.png, ...
    确保至少两位数（如01, 02, ..., 10, 11, ...）
    """
    try:
        # 切换到指定文件夹
        os.chdir(folder_path)
        
        # 获取所有PNG文件并按名称排序
        png_files = [f for f in os.listdir('.') if f.lower().endswith('.png')]
        png_files.sort()  # 按A-Z顺序排序
        
        if not png_files:
            print("文件夹中没有PNG图片文件！")
            return
        
        print(f"找到 {len(png_files)} 个PNG文件，准备重命名...")
        
        # 计算需要的数字位数，确保至少有2位数
        total_files = len(png_files)
        digits = max(2, len(str(total_files)))  # 确保至少有2位数字
        
        # 先记录原始文件名和目标文件名的映射关系
        rename_plan = []
        for i, filename in enumerate(png_files, 1):
            # 生成新的文件名，确保至少两位数，如 01.png, 02.png, ..., 10.png, ...
            new_name = f"{i:0{digits}d}.png"
            rename_plan.append((filename, new_name))
        
        # 显示重命名计划
        print("\n重命名计划：")
        for old_name, new_name in rename_plan:
            print(f"  {old_name} -> {new_name}")
        
        # 询问用户确认
        response = input("\n是否确认执行重命名？(y/n): ").lower()
        if response != 'y':
            print("操作已取消。")
            return
        
        # 执行重命名
        print("\n开始重命名...")
        renamed_count = 0
        
        for old_name, new_name in rename_plan:
            # 检查新旧文件名是否相同
            if old_name == new_name:
                print(f"  - 跳过: {old_name} 已使用目标名称")
                continue
                
            # 如果目标文件名已存在，先添加临时后缀避免冲突
            if os.path.exists(new_name):
                temp_name = f"temp_{new_name}"
                os.rename(old_name, temp_name)
                # 如果原始文件被重命名为临时文件，需要更新old_name
                old_name = temp_name
            
            # 执行重命名
            os.rename(old_name, new_name)
            renamed_count += 1
            print(f"  ✓ {old_name} -> {new_name}")
        
        print(f"\n完成！成功重命名了 {renamed_count} 个文件。")
        
    except FileNotFoundError:
        print(f"错误：找不到文件夹 '{folder_path}'")
    except PermissionError:
        print("错误：没有权限访问此文件夹。")
    except Exception as e:
        print(f"发生错误：{e}")

def wait_for_key():

    if os.name == 'nt':  # Windows系统

    else:  # Linux/Mac系统
        input("请按任意键继续...")
```

## 有关「标点符号」的话题

细心的人一路读下来可能会发现，我的标点符号（如「『引号』」、《书名号》、〈篇名号〉）的使用习惯不符合简体中文的用法。

主要还是因为我从小到大都在香港接受教育。尽管现在写字、阅读时更喜欢较为好写、简洁的简体字，但繁体字才是我更熟悉的「母字」。因此，在书面文章中使用繁体中文地区的标点写法可以说是很自然的。

其实在博客里我一直都是在使用「『』」这种引号形式。今年在写大学文章习作时被中文老师纠正后，才逐渐倾向于使用学术上更通用的写法。在简体中文中，引号多用 “ ”；而在繁体中文书面语中，则多用「」——香港、台湾都是使用这一套的哦~
